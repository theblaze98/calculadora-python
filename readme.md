# Calculadora

## ¿Que es?

Esta es una calculadora Basica creada
con __python__ en la cual se pueden realizar operaciones 
aritmeticas __basicas__.

## Funciones

Con esta caculadora se puede realizar las siguiente operacciones:

1. Suma
2. Resta
3. Multiplicación
4. division
